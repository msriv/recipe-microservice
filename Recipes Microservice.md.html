Recipes Microservice

API Model

[POST] /v1/recipe
Create a recipe item

[GET] /v1/recipe/{id}
Get a specific recipe

[GET] /v1/recipes
Get a list of all the recipes

[PUT] /v1/recipe/{id}
Update a recipe

[DELETE] /v1/recipe/{id}
Delete a recipe
